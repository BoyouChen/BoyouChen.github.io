Current Research Project

Development of deep learning algorithms for sentiment analysis in social media

My current research project involves the development of deep learning models for sentiment analysis in social media data. The goal is to develop models that can accurately detect sentiment and emotions in social media text, which can be used for a variety of applications such as customer feedback analysis and brand reputation management.

As part of my previous research project, I developed a machine translation system using neural networks. The system was trained on parallel corpora in English and Spanish, and achieved state-of-the-art performance on standard machine translation benchmarks.
